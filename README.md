# Projet de formation FreeGlisse
Etude de marché du ski occasion sur le site FreeGlisse.fr
Dans le cadre de ma formation en tant que Data Analyst, le but est d'atudier les informations du site FreeGlisse.fr pour le compte d'une entreprise qui souhaite se lancer dans le ski d'occasion.
Le projet contient un Webscrapping, un nettoyage de données avec une petite EDA, et un fichier PowerBI qui présente les données.
